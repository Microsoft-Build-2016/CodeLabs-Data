WEBSITE DEPLOYMENT

1.     Open the folder named Deployment. Go to C:\CodeLabs-Data\Module4\DeploymentError! Reference source not found.

2.     Type PowerShell into the address bar and press enter.

(This opens PowerShell with the path already at the folder).

3.     Type .\deploy.ps1 into PowerShell and press enter.

4.     In demo name, please put your initials along with Azure Pass Suffix in this format inbuildxxxx (in is your initials).

5.     Press enter.

 

6.     You will be asked to sign in using your Azure account.

7.     Type in your username and password for your Azure subscription.

8.     Click Sign in.

9.     The next process will take about 10 minutes.

10.  When it has finished, you will see some information. You should see website URL : xxxx

11.    Copy and paste this URL. This is the URL for your deployed website.

12.   Make a note of the documentDBUri and documentDbAuthKey.
